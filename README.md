# Purity-Clean

Sitio web de Purity & Clean — servicios y productos profesionales de limpieza.

## SEO implementado

### Open Graph (index.html)
- `og:type` → website
- `og:url` → https://purityclean.com/
- `og:title` → Purity & Clean | Soluciones de limpieza profesional
- `og:description` → Descripción de servicios y productos
- `og:image` → https://purityclean.com/images/og-image.jpg (1200x630)
- `og:locale` → es_CO

### Twitter Cards
- `twitter:card` → summary_large_image

### Schema.org LocalBusiness (JSON-LD)
- Datos de contacto: teléfono, email, dirección Bogotá
- GeoCoordinates: Bogotá
- OpeningHoursSpecification: L-V 08:00-18:00
- sameAs: redes sociales
- OfferCatalog con 4 servicios
- AggregateRating: 4.8 (127 reseñas)

### sitemap.xml
- URL principal con priority 1.0
- Secciones #servicios, #productos, #contacto
- Metadatos de última modificación y frecuencia

### robots.txt
- Allow general para crawlers
- Sitemap declarado
- Crawl-delay 1 para buena gestión
- Bloqueo para crawlers de SEO (AhrefsBot, SemrushBot)

### Meta tags adicionales
- canonical URL
- sitemap XML link
- keywords
- robots: index, follow

## Archivos
- `index.html` — página principal
